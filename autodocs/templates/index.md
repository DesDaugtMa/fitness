# 📝 Templates

Wiederverwendbare Vorlagen für konsistente Dokumentation.

## Verfügbare Templates

### Documentation Templates

- [[feature-template]] - Template für Feature-Dokumentation
- [[adr-template]] - Template für Architecture Decision Records
- [[test-template]] - Template für Test-Dokumentation
- [[todo-template]] - Template für TODO-Items

## Verwendung

```bash
# Feature-Template kopieren
cp autodocs/templates/feature-template.md autodocs/features/YYYY-MM-DD-feature-name.md

# ADR-Template kopieren
cp autodocs/templates/adr-template.md autodocs/adrs/adr-XXX-title.md

# Test-Template kopieren
cp autodocs/templates/test-template.md autodocs/tests/test-name.md

# TODO-Template kopieren
cp autodocs/templates/todo-template.md autodocs/todo/todo-description.md
```
